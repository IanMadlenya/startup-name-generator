# startup-name-generator

> Let's name your silly startup

```js
const name = require('@rstacruz/startup-name-generator')
name('cloud')
//=> ['Cloudary', 'Purecloud', 'Cloudlayer', 'Echocloud', ...]
```

## Install

```sh
yarn add --exact github:rstacruz/startup-name-generator
```
